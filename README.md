Food
====

Parser of On-Campus Preferred Specials

Uses the mysqlParser.py file to parse location, food category, and food item strings from websites (using pyQuery), and then sends them off to a mySQL database (using mySQLdb). Then the menu.php page grabs that information and displays it.
